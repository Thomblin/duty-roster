Library and Binary to create a schedule for a given set of people for a given set of tasks/places during a given range of dates

# How to use

Copy an example configuration file from [test/config.toml](test/config.toml) into the project directory (or where the executable is lcoated). Execute the program. It will create a csv file with the schedule.
